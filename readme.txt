== Description ==

Create a Shortcode for embeding QR Code and also enabling users with Editor Role to Checkin Attendee by scaning the QR Code.

== Installation ==

1. Download and extract CampTix QR Checkin in your `wp-content/plugins` directory
1. Activate the plugin through the Plugins menu in WordPress
1. Create a ticket or two, place the `[tix_qrcheckin]` shortcode in any camptix email template
